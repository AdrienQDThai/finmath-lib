10    /**
11    * Implementation of a time-discrete n-dimensional Brownian motion
12    * <i>W = (W<sub>1</sub>,...,W<sub>n</sub>)</i> where <i>W<sub>i</sub></i> is
13    * a Brownian motion and <i>W<sub>i</sub></i>, <i>W<sub>j</sub></i> are
14    * independent for <i>i</i> not equal <i>j</i>.
15    *
16    * For a correlated Brownian motion with see
17    * {@link net.finmath.montecarlo.CorrelatedBrownianMotion}.
18    *
19    * Here the dimension <i>n</i> is called factors since this Brownian motion is used to
20    * generate multi-dimensional multi-factor Ito processes and there one might
21    * use a different number of factors to generate Ito processes of different
22    * dimension.
23    *
24    * The quadruppel (time discretization, number of factors, number of paths, seed)
25    * defines the state of an object of this class, i.e., BrownianMotionLazyInit for which
26    * there parameters agree, generate the same random numbers.
27    *
28    * The class is immutable and thread safe. It uses lazy initialization.
29    *
30    * @author Christian Fries
31    * @version 1.6
32    * @deprecated Refactor rename. Please use BrownianMotionFromMersenneRandomNumbers instead.
33    */

39    /**
40    * Construct a Brownian motion.

42    * The constructor allows to set the factory to be used for the construction of
