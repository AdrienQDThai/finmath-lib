21    /**
22    * This class implements some numerical schemes for multi-dimensional multi-factor Ito process.
23    *
24    * It features the standard Euler scheme and the standard predictor-corrector Euler scheme
25    * for <i>Y</i>, then applies the <i>state space transform</i> \( X = f(Y) \). For the standard Euler scheme
26    * the process Y is discretized as
27    * \[
28    * Y(t_{i+1}) = Y(t_{i}) + \mu(t_{i}) \Delta t_{i} + \sigma(t_{i}) \Delta W(t_{i}) \text{.}
29    * \]
30   
31    *
32    * Hence, using the <i>state space transform</i>, it is possible to create a log-Euler scheme, i.e.,
33    * \[
34    * X(t_{i+1}) = X(t_{i}) \cdot \exp\left( (\mu(t_{i}) - \frac{1}{2} \sigma(t_{i})^2) \Delta t_{i} + \sigma(t_{i}) \Delta W(t_{i}) \right) \text{.}
35    * \]
36    *
37    * The dimension is called <code>numberOfComponents</code> here. The default for <code>numberOfFactors</code> is 1.
38    *
39    * @author Christian Fries
40    * @see MonteCarloProcess The interface definition contains more details.
41    * @version 1.4
42    */
